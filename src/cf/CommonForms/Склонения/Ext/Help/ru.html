Склонения представления выбранного объекта.

Открывается по ссылке Склонения из карточки объекта.

Склонение может быть выполнено программой без использования сервиса morpher.ru. Включить использование сервиса можно в настройках программы.

Возможны неточности автоматического склонения.

Поэтому предусмотрена возможность исправить неточные склонения и сохранить правильный вариант в карточке объекты, для того чтобы в печатные формы выводились правильные варианты склонения.

Исправьте склонения вручную, нажмите кнопку ОК. Для сохранения склонений в карточке объекта программы нажмите кнопку Записать и закрыть.

Если исправленный вариант склонения не устраивает, нажмите кнопку Просклонять повторно.
