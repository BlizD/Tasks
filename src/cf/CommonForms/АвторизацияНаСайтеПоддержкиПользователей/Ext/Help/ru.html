Для доступа к онлайн-ресурсам "1С" необходимо ввести регистрационные данные, полученные на сайте поддержки пользователей 1С:Предприятия:

  • Введите Логин;
  • Введите Пароль.
  • Для того чтобы больше не вводить эти данные в дальнейшем, включите флажок Запомнить пароль.
  • Для того чтобы получить доступ к онлайн-ресурсам "1С", щелкните по ссылке Перейти к регистрации на сайте поддержки пользователей.
  • Для авторизации на сайте поддержки и возврата в программу нажмите Продолжить.
