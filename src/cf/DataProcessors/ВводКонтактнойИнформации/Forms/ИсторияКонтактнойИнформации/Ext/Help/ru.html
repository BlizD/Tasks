Предназначена для хранения истории контактной информации, например, адреса.

История контактной информации может быть полезной, например, история изменения юридического адреса позволяет корректно печатать документы с юридическим адресом на дату ввода этого документа.

Изменение адреса

  • В окне История адреса, которое открывается с помощью гиперссылки История или в меню Еще окна Ввод адреса, можно "задним числом" изменять дату, с которой действует адрес, а также сам адрес.
  • Для этого выделите нужную строку с адресом, нажмите Изменить. После этого становится доступной кнопка  Выбрать, по которой открывается шаблон ввода адреса.
  • В окне шаблона для актуального адреса выводится сообщение Адрес действует в настоящее время.
