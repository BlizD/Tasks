Предназначена для ввода населенного пункта по детальному шаблону.

Открывается из шаблона ввода адреса в поле Город, населенный пункт по кнопке  Выбрать.

В поле Регион необходимо ввести наименование субъекта Российской Федерации, к которому относится населенный пункт. При вводе поля Регион, если адресный классификатор для этого субъекта Российской Федерации не заполнен и у текущего пользователя есть права на загрузку, будет выведено предложение загрузить классификатор. Также, при загруженном классификаторе, можно выбрать часть адреса из списка, нажав  Выбрать.

Поля Округ, Район, Город, Внутр.район, Нас.пункт предназначены для ввода соответствующих частей адреса. Поля, которые должны быть заполнены, зависят от адреса. Например, в некоторых адресах может не быть района, в других - города или населенного пункта. Если выбрано устаревшее наименование, программа выводит сообщение. Подтвердите свой выбор, нажав кнопку Да для продолжения ввода. Для того чтобы программа не показывала в списках устаревшие адреса, при настройке данного вида контактной информации включите флажок Скрывать неактуальные адреса.

После окончания ввода города или населенного пункта нажмите ОК для продолжения ввода адреса.

Если для региона адреса загружен адресный классификатор, то по мере набора в каждом поле будут предлагаться варианты соответствующей части адреса с учетом полей, заполненных ранее.
