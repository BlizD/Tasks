В данном диалоге можно ввести границы периода рассчитанных итогов для регистров накопления или бухгалтерии.

По умолчанию предлагаются следующие даты для установки границы рассчитанных итогов:

  • для регистров накопления - окончание предыдущего месяца;
  • для регистров бухгалтерии - окончание текущего месяца.

Даты можно написать вручную или заполнить с помощью кнопки .

После изменения дата автоматически преобразуется к дате окончания месяца выбранной даты.

Если команда установки периода рассчитанных итогов выбрана для одного регистра, то будет доступно изменение только одного параметра, соответствующего типу регистра.
