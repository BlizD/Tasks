В программу можно загрузить одновременно несколько выбранных файлов или несколько папок с файлами с компьютера на компьютере. Для этого необходимо выполнить соответствующие команды в списке файлов.

Если один или несколько выбранных файлов превышают предельно допустимый размер, то программа выводит предупреждение со списком файлов, которые невозможно загрузить.

Нажмите Продолжить импорт, для того чтобы:

  • загрузить остальные файлы, перечисленные в списке для загрузки, которые проходят по размеру;
  • получить отчет об ошибках с перечнем файлов, по которым импорт не состоялся.

Нажмите Отмена для возврата в программу без продолжения попыток. Если планировалось загрузить несколько файлов и другие файлы удовлетворяли условиям, они также не будут загружены в программу.

Предельно допустимый размер файлов можно увеличить в настройках программы. Для этого в разделе Администрирование - Настройки работы с файлами укажите Максимальный размер файла. По умолчанию равен 50 Мб.
