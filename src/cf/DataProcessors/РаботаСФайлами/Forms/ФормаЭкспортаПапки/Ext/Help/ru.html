Позволяет скопировать выбранную папку с файлами из программы на компьютер. При этом копируются только активные версии файлов.

Вызывается из списка файлов по команде контекстного меню папки по правой кнопке мыши Сохранить папку.

Сохранение папки

  •  Выберите на компьютере каталог,в который будет копироваться папка.
  • Нажмите Сохранить папку.
