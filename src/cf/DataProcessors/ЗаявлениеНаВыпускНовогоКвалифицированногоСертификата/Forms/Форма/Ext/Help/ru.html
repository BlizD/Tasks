Предназначена для подготовки заявления на выпуск нового квалифицированного сертификата в удостоверяющий центр 1С.

Соглашение

  • На странице прочтите соглашение о присоединении к регламенту удостоверяющего центра (УЦ). По гиперссылке можно перейти к просмотру информации об УЦ.
  • Включите флажок, для того чтобы Принять Соглашение.
  • Нажмите Далее для перехода к подготовке заявления.

Организация

  • На странице необходимо заполнить данные об организации. Выберите организацию из списка. Если данные организации были заполнены, то поля заполняются программой автоматически.
  • Программа проверяет правильность заполнения реквизитов.
  • Нажмите Далее для перехода к заполнению данных владельца сертификата.

Сохранение заявления

  • Заявление сохраняется автоматически на стадии подготовки и хранится в списке сертификатов. В любой момент к нему можно вернуться и продолжить работу.

Владелец сертификата

  • На странице необходимо заполнить данные владельца сертификата. Если данные физического лица, связанного с организацией, были введены в программу, то они будут автоматически заполнены.
  • Программа проверяет правильность заполнения полей СНИЛС и Адрес электронной почты.
  • Нажмите Далее, для того чтобы перейти к созданию ключа электронной подписи и запроса на сертификат.

Создание ключа электронной подписи

  • На странице проверяются данные владельца электронной подписи. С помощью ссылок можно получить дополнительную информацию.
  • В поле С помощью программы выберите программу - криптопровайдер из списка установленных на компьютере.
  • Нажмите Далее, для того чтобы создать ключ электронной подписи и запрос на сертификат.
  • Состояние подготовки заявления меняется на Подготовлено.

Отправка заявления

  • На этой странице необходимо заполнить сведения о руководителе для печати заявления.
  • Также необходимо подготовить комплект документов:
    • Заявление на выпуск сертификата (нажмите Печать для распечатки заявления);
    • Копия свидетельства о постановке на учет в налоговом органе (ИНН);
    • Копия свидетельства о государственной регистрации индивидуального предпринимателя (ОГРН);
    • Копия документа, удостоверяющего личность владельца сертификата;
    • Копия страхового свидетельства обязательного пенсионного страхования (СНИЛС) владельца сертификата.
  • Заполните Данные обслуживающей организации (партнера 1С).
  • Для отправки заявления на обработку нажмите Далее.

Ожидание обработки заявления

  • После отправки заявления открывается страница Ожидание обработки заявления. Для проверки состояния нажмите кнопку Обновить состояние.

Установка полученного сертификата

  • После получения сертификата установите его на компьютер с помощью кнопки Установить сертификат.
  • Воспользуйтесь соответствующей ссылкой, для того чтобы Выгрузить сертификат.
