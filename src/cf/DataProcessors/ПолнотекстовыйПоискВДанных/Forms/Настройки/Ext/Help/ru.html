Предназначено для настройки области полнотекстового поиска.  

Можно ограничить область полнотекстового поиска с помощью переключателя, тем самым увеличив точность результатов поиска.

По умолчанию поиск производится Везде.

Ограничение области поиска

  • С помощью переключателя укажите В разделах.
  • Выберите разделы и списки, в которых будет производиться поиск, с помощью флажков.
  • Нажмите ОК.
