Предназначена для выбора устройства сканирования.

Устройство сканирования должно быть подключено к компьютеру. Выберите наименование устройства из  программе имеется список возможных устройств сканирования, она предлагает выбрать из выпадающего списка.

Нажмите кнопку Выбрать, для того чтобы подтвердить свой выбор.
