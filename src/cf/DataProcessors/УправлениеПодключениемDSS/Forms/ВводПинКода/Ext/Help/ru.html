Предназначена для ввода пин-кода доступа к закрытому ключу сертификата при выполнении операции сервиса DSS.

В типовом сценарии запрос пин-кода осуществляется для первой операции на сервисе DSS с выбранным сертификатом и при выполнении последующих операций данное окно не отображается, т.к. пароль сохраняется на время сеанса работы пользователя в программе.

Ввод пин-кода доступа к закрытому ключу сертификата

  • В поле сертификат указываются данные сертификата, с помощью которого выполняется операция.
  • В поле пин-код необходимо ввести пин-код доступа.
  • Флажок сохранить позволяет сохранить введенный пароль в базе, в случае успешного подключения к сервису. В последующих подключениях программа сразу пытается авторизовать с указанным сертификатом, без показа этой форм.
  • Ссылка сменить пин-код вызывает отдельный процесс смены пин-кода для выбранного сертификата. В случае успешной смены новый пин-код сразу используется для доступа к операции.
