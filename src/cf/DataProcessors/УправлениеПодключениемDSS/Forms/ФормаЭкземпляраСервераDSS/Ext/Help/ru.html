Предназначена для хранения настроек подключения к сервису DSS.

Сервис DSS позволяет хранить сертификаты и закрытые ключи к ним. Это позволяет пользователю использовать функции подписания, шифрования, расшифровки данных практически с любого компьютера. Для использования этого функционала не требуется установки криптопровайдеров и сертификатов на локальный компьютер, достаточно предоставить данные для авторизации на настроенном сервисе.

К каждому серверу DSS привязываются учетные записи этого сервиса, указываемые в справочнике «Учетные записи DSS».

Настройка подключения

  • В поле адрес сервера указывается полный веб-адрес подключения к сервису.
  • В поле адрес сервиса центра идентификации указывается полный или относительный адрес подключения. 
  • В поле адрес сервиса электронной подписи указывается полный или относительный адрес подключения. 
  • В поле адрес сервиса обработки документов указывается полный или относительный адрес подключения.
  • В поле адрес службы проверки подписи и сертификатов указывает полный или относительный адрес подключения. 
  • В поле адрес личного кабинета указывается полный или относительный адрес подключения. Это поле является не обязательным, информация в этом может быть удобной для организации быстрого перехода к странице личного кабинета пользователя опубликованного не сервисе.
  • В поле идентификатор клиента указываются ключевые данные для идентификации пользователя.
  • Поле идентификатор сервиса может заполняться автоматически или указано пользователем вручную.
  • По кнопке Расширенные настройки можно перейти на отдельную форму для указания необязательных настроек:
    • Представление сервера.
    • Время ожидания подключения к сервису (таймаут) указывает в секундах.
    • Версия поддерживаемого АПИ.
    • Отключить использование данного сервера – исключается из последующего создания учетных записей, и выбора учетных записей.
  • По кнопке Ограничение авторизации можно перейти на отдельную форму, позволяющую задать список способов авторизации, доступных этому серверу.
  • Также по ссылке Учетные записи DSS можно перейти к списку зарегистрированных учетных записей DSS, связанных с этим элементом сервера DSS.
