Предназначена для настройки параметров обновления программы через Интернет.

Для обновления с пользовательского сайта необходимо указать код пользователя и пароль для доступа. Доступ к получению обновлений имеют только зарегистрированные пользователи программ 1С:Предприятие. Подробная информация о том, как можно получить код пользователя и пароль, содержится в разделе методической поддержки на сайте v8.1c.ru.

Открывается по команде Настройка обновления программы через Интернет из раздела Администрирование - Поддержка и обслуживания - Обновление версии программы.

С помощью переключателя установите автоматическую проверку обновлений:

  • Отключена;
  • При запуске программы;
  • По заданному расписанию - при выборе этого значения заполните расписание автоматической проверки обновлений. Для этого перейдите по ссылке, которая принимает значение Не задано или показывает ранее введенное расписание.

Заполните параметры доступа на пользовательский сайт v8.1c.ru:

  • Код пользователя;
  • Пароль.

При необходимости включите флажок Запомнить пароль. Перейдите по ссылке Как получить код пользователя и пароль, если у вас еще нет возможности обновления программы через интернет.
