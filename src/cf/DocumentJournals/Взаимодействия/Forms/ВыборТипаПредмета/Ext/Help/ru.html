Предназначена для выбора типа предмета в документах взаимодействий.

Для заполнения поля Предмет сначала выберите из списка тип предмета. После этого откроется список выбранных объектов программы.

По умолчанию в списке отображаются типы документов взаимодействий.

Для отображения других типов объектов программы включите флажок Не отображать взаимодействия.
