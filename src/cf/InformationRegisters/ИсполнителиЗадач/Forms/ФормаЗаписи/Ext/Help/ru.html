Сведения об исполнителе задач при ролевой адресации.

Открывается по команде Создать из списка Исполнители роли.

Ввод исполнителя роли

  • С помощью кнопки выберите тип списка - Внешние пользователи или Пользователи.
  • Выберите Исполнителя роли из соответствующего списка.
  • В поле Выполняет роль выберите роль исполнителя задач из списка (может быть выбрана программой по умолчанию).
  • Для роли может потребоваться выбрать из списка основной и дополнительный объекты адресации.
