Основной рабочий каталог - специальный каталог на Вашем компьютере, в котором программа размещает просматриваемые и редактируемые Вами файлы. Список предназначен для управления файлами в рабочем каталоге.

Вызывается из персональных настроек программы (закладка Работа с файлами) по команде Настройка основного рабочего каталога - Открыть список файлов.

В списке выводится:

  •  Файл занят для редактирования - флажок включен, если файл редактируется.
  • Файл - формат (выводится в виде значка) и наименование файла;
  • Размер (Кб);
  • Дата изменения.

С помощью команды Еще - Изменить форму можно добавить в список дополнительные поля.

Доступные действия

  • Для удаления ненужных файлов из рабочего каталога нажмите Удалить. При этом будут удалены только те файлы, в которые Вами не вносились какие-либо изменения.
  • Для того чтобы сохранить изменения в файле и освободить его для других пользователей, нажмите Закончить редактирование.
  • Для того чтобы освободить занятые Вами файлы, используйте команду Отменить редактирование.
  • Нажмите Открыть каталог файла, для того чтобы просмотреть папку, в которой хранится редактируемый файл.
  • Выполните соответствующую команду контекстного меню по правой кнопке мыши или меню Еще, для того чтобы Открыть карточку файла.
