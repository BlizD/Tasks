Предназначен для ведения учетных записей электронной почты, которые используются для получения и отправки электронных сообщений.

Список можно открыть соответствующей командой в разделе Администрирование - Органайзер. 

В списке выводится:

  • Наименование почты;
  • Имя пользователя - указывается имя отправителя писем, оно может отличаться от имени пользователя в программе;
  • Адрес электронной почты в формате name@server.net. 

Для учетных записей, получение и отправка с которых выполняется автоматически без участия пользователя, должен быть обязательно введен пароль. Также для корректной работы программы обязательно должна быть настроена системная учетная запись электронной почты.

Системная учетная запись электронной почты

  • Программа поставляется с предопределенной настройкой электронной почты, которая предназначена для рассылки различных уведомлений из программы. Например, для автоматической отправки сообщений администратору по завершению обновления программы. Для этого предварительное необходимо настроить ее.
  • Настройка основной почты для рассылок уведомлений производится по соответствующей команде в разделе Администрирование - Органайзер. Также можно Изменить ее в списке.

Настройка электронной почты

  • При попытке сохранения программа проверяет введенные учетные записи электронной почты (пытается отослать пробное сообщение). Учетные записи, не прошедшие проверку, не сохраняются.
  • Нажмите Создать, введите необходимые поля:
    • Укажите Адрес почты в формате name@server.net;
    • С помощью переключателя выберите способ авторизации:
      • Ввести пароль, в этом случае укажите в поле справа Пароль почты;
      • Авторизоваться на почтовом сервисе, в этом случае программа выводит инструкцию авторизации, которая зависит от почтового сервиса. Следуйте инструкции программы.
        • На почтовом сервисе потребуется указать данные о приложении, например, callback URL, который программа формирует автоматически.
        • Воспользовавшись указаниями почтового сервиса, зарегистрируйте приложение, получите данные для авторизации и заполните нужные поля, например: 
              • ID приложения;
              • пароль приложения (client secret).
    • Установите с помощью флажков, как Почта будет использоваться:
      • Для отправки писем;
      • Для получения писем;
    • Укажите с помощью переключателя, кем будет использоваться почта:
      • Всеми пользователями (общая почта)
      • Только мной.
    • Укажите Имя отправителя (Ваше имя или название организации);
  • Нажмите Создать. Будет предпринята попытка автоматической настройки и проверки почты. Следуйте указаниям программы. При этом если при автоматической настройке произошел сбой, можно попытаться настроить параметры вручную. 
  • Если автоматически настроить почту не удалось, нажмите Настроить вручную.

    • Нажмите Далее для заполнения остальных параметров, при этом программа проставляет рекомендуемые параметры для большинства почтовых серверов. Их требуется только проверить и попытаться сохранить учетную запись заново.

Настройка электронной почты

  • Нажмите  Изменить, с помощью программы произведите необходимые настройки.

Проверка настроек электронной почты

  • Выделите учетную запись в списке, нажмите Еще - Проверить почту для проверки введенных параметров. При этом отправляется стандартное почтовое отправление и проверяется возможность соединения с сервером входящих почтовых сообщений (если учетная запись используется для получения почты).
  • Если в почте не указан пароль, он будет запрошен.
