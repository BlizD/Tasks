Предназначен для просмотра задач, созданных по объекту программы.

Открывается из объекта программы по команде Задачи панели навигации.

Красным цветом обозначаются просроченные задачи. Выполненные задачи становятся неактивными, выделяются серым цветом.

В списке выводится:

  •  - важность, принимает значения Обычная (по умолчанию), Низкая, Высокая. Для того чтобы выделить бизнес-процесс из списка, или наоборот, указать, что выполнение этого бизнес-процесса может быть отложено, выбираются соответствующие значения этого поля.
  •  - остановленные бизнес-процессы в этой колонке обозначаются значком . Для того чтобы Остановить или Продолжить бизнес-процесс, воспользуйтесь соответствующими кнопками.
  • Задача - полное наименование задачи. Рядом проставляется значок  Задача.
  • Исполнитель задачи;
  • Срок - дата и время, к которому Задание должно быть выполнено.

Просмотр задачи

  • Нажмите Изменить, для того чтобы открыть задачу для просмотра или редактирования.
  • Также можно использовать двойной щелчок левой кнопкой мыши.
