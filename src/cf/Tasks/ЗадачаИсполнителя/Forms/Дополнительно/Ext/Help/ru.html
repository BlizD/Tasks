Предназначена для просмотра основных сведений о задаче (редактирование недоступно).

Открывается из окна задачи по команде Еще - Дополнительно.

Сведения о задаче

перечислены общие данные для всех типов задач, такие как:

  • Задача - наименование задачи;
  • Номер - проставляется программой автоматически;
  • Записана - дата и время создания задачи, проставляется программой автоматически;
  • Срок - дата и время, до которого необходимо выполнить задачу.

Задача адресована

  • Исполнитель - пользователь программы, назначенный исполнителем;
  • Роль исполнителя - выбирается из списка Роли и исполнители;
    • Основной и Дополнительный объекты адресации - сведения об объектах программы, которые уточняют ролевую адресацию.

    Бизнес-процесс

    • Бизнес-процесс  - данные о бизнес-процессе, (наименование, дата  и вид), к которому принадлежит задача;
    • текущая Точка маршрута бизнес-процесса (точка карты маршрута бизнес-процесса, например, Выполнить или Проверить).

    Результат выполнения задачи

    Реквизиты этой группы заполнены, если задача выполнена:

    • флажок Выполнена
    • Выполнена - дата и время выполнения задачи.
