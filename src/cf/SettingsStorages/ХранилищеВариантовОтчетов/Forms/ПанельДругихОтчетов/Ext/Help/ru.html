Предназначена для перехода к близким по смыслу вариантам отчетов.

Открывается по команде Другие отчеты из подменю  Варианты отчета.

В списке в виде ссылок выводятся близкие по смыслу варианты отчетов, если они имеются в программе.

Открывается список остальных вариантов отчетов данного раздела. Для перехода к нужному варианту отчета нажмите на соответствующую ссылку.

В нижней части окна по умолчанию включен флажок Закрывать это окно после перехода к отчету, для того чтобы окно не мешало работе с отчетом.
